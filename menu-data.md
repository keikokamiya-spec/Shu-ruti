## 黒板から読み取れるブランド・コンセプト情報

| 要素 | 内容 |
|---|---|
| 価格帯 | 600円〜1,500円。バーとしては良心的な料理価格 |
| 料理ジャンル | 和（お好み焼・梅水晶・餃子）×洋（プロシュート・ペペロンチーノ・パスタ）の混在 |
| こだわり表現 | 「俺の！特製」「低温調理した しっとり」「ミシュラン御用達！」など個性的なネーミング |
| フルーツカクテル | 季節感のあるフルーツカクテルを提供（バーとしての差別化） |
| ◎マーク | あまおうアイスに特別マーク → 自信のある一品として積極推薦 |

## HP掲載時の推奨表現（Menuセクション）

黒板メニューの写真をそのまま見せるスタイルが世界観に合う。
以下のコピーで添えると効果的：

```
「料理も、バーを超える。」
バーとは思えないほど豊富なフードメニュー。
季節のフルーツカクテルから、こだわりの一皿まで。
```

または黒板写真 + メニュー名リストの組み合わせで構成する。
